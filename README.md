## Installation and Setup Instructions

Clone repository and follow the steps below.

**First Time Setup**

`make setup`

**Start Application**

`make up`

Navigate to `http://localhost:3000/`

**To Run Tests**

`make test.unit`

**To Run Coverage**

`make test.coverage`
